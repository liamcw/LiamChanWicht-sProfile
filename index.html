PROJECT

Smart Mirror


I've recently been fascinated with the home becoming more and more tech-infused with the Internet of Things (IoT).

The only problem for me and many others is that it is difficult to set up a smart home that is suited for renters. As a renter, you are generally unable to affix anything to walls, install thermostats, or do anything else to compromise the integrity of the building you live in. This project is not as ambitious as creating life-saving robots or programs that can change whole neighbourhoods but it is one that is interesting to me.

Every morning when I get ready, I jump out of the shower and look in the boring old mirror, thinking about what needs to be done for the day. If I spend too much time grooming and pondering, the time goes by and I have the potential to be late!

I hear what you're saying - "Just check your phone!".

Well right out of the shower, I don't want to be checking my phone as my hands are wet and most of the time they have that gross pruny texture that doesn't let me activate my fingerprint sensor. What if all my information was right there, where I was already looking, ready and waiting for me?

The time! A list of updateable daily objectives/meetings/tasks! The weather! What if I could speak to my mirror like a personal assistant? What if it was motion activated? How futuristic would that be?

Pretty damn futuristic if you ask me.

With the time displayed on the mirror, tracking how long it is taking to get ready will be breeze! Shorts or pants? Showing the weather will help in deciding how to dress for the day at a glance. A calendar or a list of daily objectives will help prepare for the day to come - an invaluable time saving tool! If these features are voice/ motion controlled, it will make navigation much easier as there won't be another interface to interact with to make it work.

There are (as of 2017) 24.6 million people living in Australia ALONE who more than likely, own some sort of shiny surface that isn't living up to its full potential! So how do we go about 'smartening' our mirrors to display all of this wonderful information?

Two words. Raspberry Pi.



Raspberry Pi


"The Raspberry Pi is a low cost, credit-card sized computer that plugs into a computer monitor or TV, and uses a standard keyboard and mouse. It is a capable little device that enables people of all ages to explore computing, and to learn how to program in languages like Scratch and Python. It’s capable of doing everything you’d expect a desktop computer to do, from browsing the internet and playing high-definition video, to making spreadsheets, word-processing, and playing games.

What’s more, the Raspberry Pi has the ability to interact with the outside world, and has been used in a wide array of digital maker projects, from music machines and parent detectors to weather stations and tweeting birdhouses with infra-red cameras. "

from What is a Raspberry Pi? https://www.raspberrypi.org/help/what-%20is-a-raspberry-pi/

A compact computer that is easily programmable and hidden behind a mirror! The Raspberry Pi is the perfect tool to achieve this goal. More specifically, the Raspberry Pi 3 Model B, as it has easy accessibility and an easy to set up wireless option. Another option would be to use Arduino, a simpler platform by all definitions. I believe the Raspberry Pi would be the better choice as it runs Linux and is much more sophisticated (more RAM, more ports, and a whole operating system!). A USB mouse and keyboard will also be needed to interact with the Pi.

Monitor


So we have what will process all of our lovely ideas but how will we display it? A computer monitor. Plugging in the Raspberry Pi to even a second-hand computer monitor will allow us to have a Graphic User Interface (GUI) to play around with. As the Pi has a HDMI out port, we will need a monitor that at least has a HDMI In. Speaking of HDMI ports, quite obviously we will need a HDMI cable so add that to the shopping list. We will also need a Micro HDMI Cable for the Pi Power Supply. An added advantage of this project is the potential 'upcycling' of an old monitor that is just laying about. If there isn't one handy, cheap monitors are a dime a dozen nowadays.

Mirror


Now unfortunately not all mirrors will be the most optimal choice for this project. One way mirrors will work best for this project as they allow the glass to be semi-transparent to be able to 'shine through' and display 'on' the mirror. Think of all those interrogation scenes on TV. When one room is lit, it behaves like a mirror. Otherwise, it's just a normal window. Ideally, the mirror will be the same size if not a little larger than the monitor. There is the possibility of having a smaller 'smart area' with larger mirror space by adjusting the ratios of monitor to mirror.

Software


This is where you discover that this fantastic idea has been around for about 4 years already. The upside of this, is that you can just use the open source software, MagicMirror². The downside is that you won't be internationally famous for thinking you came up with a cool idea.

MagicMirror² is an open source modular smart mirror platform. With a growing list of installable modules, the MagicMirror² allows you to convert your hallway or bathroom mirror into your personal assistant.
from MagicMirror² GitHub Repo https://github.com/MichMich/MagicMirror

This repo contains a complete tutorial covering installation and customisation. Nice. No need to code from scratch! This cuts a lot of hours out of the project that can be used in other endeavors. Not only that, but when you are comfortable enough with the code, you can contribute to it through GitHub's collaborative systems.

Conclusion


This project can be completed within a single weekend and I believe it is a very feasible option to add another smart device to your collection! The ease of access to all components really help make this project accessible to those of all skill levels. The level of documentation of not only the MagicMirror² platform but the Raspberry Pi will make the process even easier for people who have zero experience but still want to undertake the project.

I believe this project will not only solve the problem of a lack of 'at glance updates' but also serve as a good introductory activity for those interested in the Raspberry Pi and its capabilities. Personally, I will be attempting this project in the near future as it will teach me how to interact with hardware components and its interaction with software. Plus, i'll get a cool future mirror out of it.
